## Pre-requisites ✅

- Node.js must be installed on your machine.
- JavaScript knowledge is required.
- Basic knowledge of React is required.

## Installation 🛠️
If you want to run this project locally, follow the below steps:

1. Installation
 ```
 cd frontend
 npm install
 ```
2. Run
```
npm run dev
```
